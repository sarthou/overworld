Overworld Tutorials

Overworld comes with a series of tutorials to help new users become familiar with the its use. Overworld initial use is mainly base on configuration and description. Over these tutorials we will see step by step how to setup a robot, objects, and humans. With the last tutorial, we will see how you can easily adapt overworld for your specific robot.

As Overworld is based on an ontology to describe all the entities, ff you do not know what an ontology is, you can take a look at this good site.

Before starting these tutorials, make sure that you have every thing properly installed. To do so follow the installation page.

Each tutorial is based on the previous one so follow them in the order!
